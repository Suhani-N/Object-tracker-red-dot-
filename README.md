# Red Object Tracker
This is a simple computer vision project that detects and tracks red-colored objects in real-time using OpenCV.

## Features
- It detects red objects using HSV color space
- Also draws bounding box around detected object
- A real-time tracking using webcam

## Tools/Tech Used
- Python
- OpenCV

## How to run
1. Install OpenCV
2. Run the Python file
3. Run the code in terminal(VScode)
4. Show a red object to webcam
